# Open Design catalog — index

One row per vendored design system. `pool` is lila's curation layer (not upstream):
**default** = the ~3 safe neutrals the framework may draw blindly; **browsable** = the curated
slice the selection skill offers on request; **full** = vendored verbatim, reachable only by an
explicit `LILA_DESIGN=<brand>` pin. Pools nest: default ⊂ browsable ⊂ full.

Generated from the vendored `DESIGN.md` files — see [PROVENANCE](PROVENANCE) for the upstream commit.
